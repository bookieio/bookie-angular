## Installation

```
$ git clone https://github.com/bookieio/bookie-angular.git
$ cd bookie-angular
$ npm install
```

## Development

```
$ npm install -g grunt-cli
$ grunt
```

Go to [http://localhost:9001](http://localhost:9001)
